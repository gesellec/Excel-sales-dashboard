# Excel Sales Dashboard

## Overview
This project is an interactive Excel dashboard that analyzes sales performance by product and region.

## Files
- `pelican-stores-analysis.xlsx` - Excel dashboard
- `analysis.md` - Detailed data analysis and insights

# Skills Used
- Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Analysis

## Key Insights
- Sales are driven by a small number of high-performing products
- Revenue varies across regions
- Data can be explored interactively using slicers

## Summary
This project demonstrates how raw data can be transformed into meaningful insights using Excel.
